Fake News Detection System

Technologies Used:
- Python
- Flask
- Machine Learning (TF-IDF + PassiveAggressiveClassifier)

Steps to Run:

1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python train_model.py

3. Run the web application:
   python app.py

4. Open browser:
   http://127.0.0.1:5000/

5. Enter news text and check result.
